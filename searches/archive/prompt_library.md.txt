# Job Search Prompt Library

*Organized collection of reusable search prompts for job hunting automation*

---

## Quick Start

**To run a search:**
1. Pick a prompt from categories below
2. Click file link to view full prompt
3. Customize: roles, locations, dates
4. Copy/paste into CoWork
5. Attach resume if scoring enabled
6. Run!
