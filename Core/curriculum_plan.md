# 12-Week AI Automation Curriculum

## Overall Goal

Build practical automation skills for startup operator roles through real projects.

## Curriculum Structure

**Weeks 1-8: Deep Mastery (Claude/CoWork)**

- Week 1: ✅ Foundations, first automation, prompt engineering basics
- Week 2: ✅ One-click automation, Git basics, multi-board search, friend template system
- Week 3: 🔄 Multi-user operations, data consolidation & analysis, professional deliverables
  - Session 1: ✅ Friend search operations (Aaron Week 2, Phil template rebuild, Vivienne onboarding)
  - Session 2: Job tracking system (application status, pipeline stages)
  - Session 3: CSV → formatted .xlsx conversion (filters, sorting, conditional formatting)
  - Session 4: Analysis → professional PDF (tables, charts, executive summaries)
  - Session 5: Email template integration (tracking data in weekly reports)
- Week 4: Application tracking & pipeline management
- Week 5: Advanced scoring, pattern recognition & search optimization
- Week 6: Claude Skills — packaging workflows into reusable instruction sets
- Week 7: Claude Skills — multi-step automations, error handling
- Week 8: Production polish, edge cases, scaling

**Weeks 9-10: Strategic Survey**

- Week 9: Claude Code (command-line developer tools)
- Week 10: Landscape survey (Zapier/Make/n8n, Cursor/Copilot)

**Weeks 11-12: Integration & Portfolio**

- Week 11: Transfer skills to new tool
- Week 12: Portfolio polish, go public

## Decision Rationale

Chose depth-first over breadth because:

- Depth transfers, breadth doesn't
- 12 weeks insufficient for both
- Startup roles need builders, not tool collectors
- Fundamentals apply across tools
